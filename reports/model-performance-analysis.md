### 分析

**改善的核心原因**：

- **信號品質提升**：分位數 spread 從 6.5 bps → 10.3 bps（+59%），且 t-stat 從不顯著（1.25）變為顯著（2.30）。DoubleEnsemble 的 sample reweighting 讓模型更聚焦於「可區分」的樣本，permutation-based feature selection 過濾了噪音因子。
- **分數排名更穩定**：Kendall tau 0.477 → 0.494，Top-10 overlap 49.1% → 51.0%，降低了不必要的換手。
- **完美單調性**：rho = 1.00 表示分數高低與實際報酬完全正相關（分位數層面），LightGBM 時期的 0.90 表示有部分分位序被顛倒。

**仍存在的問題**：

- **IC 水準未改善**：Backtest IC 從 0.0107 → 0.0108，幾乎一樣。這符合預期——DoubleEnsemble 改善的是穩定性和排名品質，不是絕對預測精度。
- **IC Decay 仍然偏高**：71.4% 雖比 78.5% 好，但遠高於文獻建議的 < 50%。Valid IC 降低（0.0498 → 0.0376）說明模型不再過度擬合訓練資料，這是正面訊號。
- **Valid-Backtest 相關仍為負**：-0.039 表示訓練時表現好的模型，實盤不一定好。
- **2024 年表現劣化**：超額從 +12.4% 降到 -8.4%，需要關注。

| 指標 | LightGBM | DoubleEnsemble | 評價 |

| Avg Backtest IC | 0.0107 | 0.0108 | 實盤 IC 持平 |
| **IC Decay** | **78.5%** | **71.4%** | 改善 7pp，但仍偏高 |
| Valid-Backtest Corr | -0.045 | -0.039 | 仍為負相關 |
| Backtest IC > 0 模型% | 56.6% | 56.3% | 持平 |
| 模型總數 | 106 | 151 | DoubleEnsemble 涵蓋更多週 |

### 改善評估

IC Decay 從 78.5% 降到 71.4%，改善幅度 7 個百分點。這**低於預期**（原先期望降到 40-60%）。

但有兩個正面訊號：
1. **Valid IC 主動降低**：0.0498 → 0.0376（-24%），說明模型不再「記住」訓練資料
2. **Backtest IC 維持不變**：0.0108 vs 0.0107，實盤預測能力沒有因為 Valid IC 降低而下降

這表示 DoubleEnsemble 的 feature selection 確實在過濾噪音因子，只是過擬合問題的根源可能不完全在模型層面——更可能在於市場 regime 變化速度超過模型更新速度。
